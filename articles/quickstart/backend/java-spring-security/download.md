To run it from the command line:

```bash
# In Linux / macOS
./gradlew clean bootRun
# In Windows
gradlew.bat clean bootRun
```

The sample includes a [Docker](https://www.docker.com) image ready to run with the following command:

```bash
# In Linux / macOS
sh exec.sh
# In Windows' Powershell
./exec.ps1
```
