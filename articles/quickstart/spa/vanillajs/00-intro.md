## Dependencies

To integrate your JavaScript application with Auth0, add the Lock widget. You can get it from Bower or from the Auth0 CDN.

**Bower**

```bash
bower install auth0-lock
```

**CDN**

```html
<script src="${lock_url}"></script>
```
