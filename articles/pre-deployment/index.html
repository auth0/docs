Production Checks

Before you go live, run Auth0's production checks suite to ensure that your tenants are ready for use in a production environment.

  • How to Run the Production Checks

    Run the production checks against one or more applications to see if they're production-ready or not.

  • Production Check Results

    Production checks fall into the following categories:

    • Required Fixes
    • Recommended Fixes
    • Fixes to Comply with Best Practices
  • How to Read Your Results Set

    View your results to see any outstanding required and recommended tasks, as well as changes you can make to ensure that you comply with best practices.
